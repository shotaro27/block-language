# ブロックで可視化した英語の構造

## 概要

Google Blocklyを使用してビジュアルプログラミング言語のように英語を組み立てられるソフトウェアを制作した。
![スクショ](/img1.png) 

単語を並べて「表示」ボタンを押すと、画像のように英語の文章になって表示される。

## 内容

- index.html

## 実行方法

- Blocklyをダウンロードしてフォルダ直下に入れる
- index.htmlを開く

## 必要なソフトウェアとバージョン

|   ソフトウェア  |  バージョン  |
|:--------------|:---------------|
| Google Blockly| Q4 2021 Patch 4|
